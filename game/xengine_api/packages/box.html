xengine.RBBox(圆形包围盒)
属性 说明 示例
x 中心坐标x(继承自包围盒抽象类)
y 中心坐标y(继承自包围盒抽象类)
c 颜色(继承自包围盒抽象类)
r 包围盒半径
方法 说明 示例
init(x,y) 初始化圆形包围盒，x,y是圆形中心坐标
collided(tBox) 碰撞检测，返回布尔值。参数tBox是一个含向量及半径的对象，如tBox.x,tBox.y,tBox.r
show(ctx) 把包围盒显示在画布中

xengine.ABBox(矩形包围盒)
属性 说明 示例
x 中心坐标x(继承自包围盒抽象类)
y 中心坐标y(继承自包围盒抽象类)
c 颜色(继承自包围盒抽象类)
w 宽度
h 高度
方法 说明 示例
init(x,y,w,h) 初始化矩形包围盒，x,y是矩形中心坐标
collided(tBox) 碰撞检测，返回布尔值。参数tBox是一个含向量及宽高的对象，如tBox.x,tBox.y,tBox.w,tBox.h
show(ctx) 把包围盒显示在画布中

xengine.PBBox(凸多边形包围盒)
属性 说明 示例
x 中心坐标x(继承自包围盒抽象类)
y 中心坐标y(继承自包围盒抽象类)
c 颜色(继承自包围盒抽象类)
方法 说明 示例
init(x,y,pArr) 初始化矩形包围盒，x,y是多边形中心坐标,pArr是一个顶点数组，点的坐标采用相对中心点坐标，按顺时针存放各顶点
collided(tBox) 碰撞检测，返回布尔值。
show(ctx) 把包围盒显示在画布中
