xengine.State(状态抽象类)
属性 说明 示例
ctx 保存的this指针
type 状态类型
方法 说明 示例
change() 调用context changeState方法转换到其他状态
update() 当前状态的活动
enter() 退出状态执行

xengine.StateContext(状态管理类)
属性 说明 示例
owner 和该状态类关联的对象
states 状态对象
currState 当前的状态
方法 说明 示例
change(type) 转换状态
get(type) 获取状态
add(state) 注册一个状态类
remove(type) 销毁一个状态类
clear() 情况状态类
update() 该方法在游戏主循环中使用
