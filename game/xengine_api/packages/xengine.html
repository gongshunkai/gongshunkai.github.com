xengine.game(游戏主体类)
方法 说明 示例
run(fun,fps) 运行游戏，第一个参数可以传一个方法，第二个参数是游戏的fps，如果不传值该参数默认是60

xengine.game.run(function(){

　　//游戏的主逻辑代码....

},30);
pause() 暂停游戏 xengine.game.pause()
resume() 继续游戏 xengine.game.resume()
stop() 终止游戏 xengine.game.stop()
addListener(fn) 添加监听器 xengine.game.addListener(fn)
clearListener() 清空监听器列表 xengine.game.clearListener()

xengine.Scene(游戏场景类)
属性 说明 示例
x 场景的x坐标
y 场景的y坐标
w 场景的宽度
h 场景的高度
color 场景的背景颜色
cvs 绑定的canvas元素
ctx canvas环境
rObjs 所有的渲染对象
namedRObjs 命名的渲染对象，便于根据名称快速查找对象
方法 说明 示例
addChild(obj) 把一个渲染对象添加到场景中

var Scene = xengine.Scene.extend({

　　init(options){

　　　　this._super(options);

　　　　this.addChild(obj);

　　}

});
removeChild(obj) 删除对象

var sc = new xengine.Scene();

sc.removeChild(obj);
clearAll() 清除所有渲染对象

var sc = new xengine.Scene();

sc.clearAll();
resize() 自适应屏幕大小，可以在初始化场景的函数中，以及浏览器大小发生改变时调用此方法
getCvsRect() 返回一个包含画布坐标及尺寸的对象

var cr = sc.getCvsRect(),

　　x=cr.x,

　　y=cr.y,

　　w=cr.w,

　　h=cr.h;

clear() 清除canvas背景
show() 显示场景
hide() 隐藏场景
fadeOut() 场景淡出效果
fadeIn() 场景淡入效果
addListener(fn) 添加监听器 sc.addListener(fn)
clearListener() 清空监听器列表 sc.clearListener()

xengine.director(游戏场景管理类)
属性 说明 示例
namedScenes 以命名方式保存,便于快速通过名称获取场景 xengine.director.namedScenes
scenes 以堆栈方式保存所有场景，最后的元素为栈顶 xengine.director.scenes
方法 说明 示例
push(scene) 把一个游戏场景压入游戏场景管理类的堆栈中 xengine.director.push(scene); //创建了一个游戏场景必须将它压入游戏场景管理堆栈中才会在画布中显示出来，否则即使创建了场景也看不到。
pop() 移除顶部场景 xengine.director.pop()
remove(scene) 删除场景 xengine.director.remove(scene)
swap(from,to) 交换场景位置 xengine.director.swap(from,to)
bringToFirst(scene) 把某个场景移动到最顶部 xengine.director.bringToFirst(scene)
bringToLast(scene) 把某个场景移动到最底部 xengine.director.bringToLast(scene)
back(scene) 场景后移 xengine.director.back(scene)
forward(scene) 场景前移 xengine.director.forward(scene)
getCurrentScene() 获取当前场景,顶部场景为当前场景 xengine.director.getCurrentScene()
clearAll() 清除所有场景 xengine.director.clearAll()

xengine.Sprite(游戏精灵类)
属性 说明 示例
owner 拥有者,指向场景对象
x 精灵的x坐标
y 精灵yx坐标
x 精灵的x坐标
w 精灵的宽度
h 精灵的高度
dx 精灵的x方向的速度
dy 精灵的y方向的速度
vx 精灵的x方向的加速度
vy 精灵的y方向的加速度
deg 精灵的角度
zIdx z-index,数字越小越先渲染
isVisible 是否可见
canRemove 是否可移除
anims 帧动画集合对象
isXFlip 是否水平反向
isYFlip 是否垂直反向
scaleX x缩放比例
scaleY y缩放比例
bBox 包围盒
tags tagPoint
方法 说明 示例
moveTo(x,y) 设置位置
move(xOff,yOff) 移动
moveStep() 移动一小步
rot(deg) 旋转deg度
update() 更新方法，每一帧调用
render(ctx) 渲染方法，每一帧调用,ctx是canvas环境
isMouseIn() 判断鼠标当前坐标是否在当前渲染对象区域中
setAnims(animations,currAnimName) 设置帧动画集合对象
addAnim(name,frames,isCurrent) 设置动画集
removeAnim(name) 删除指定名称动作
setCAnim(name) 按名称设置当前动作
setAnimSpeed(sp) 设置动画速度
getAnim(name) 按名称获取动作
getCurrentAnim() 获取当前运行的动画
getCurrFrame() 获取当前运行帧
getNextFrame() 获取下一帧信息
offScreenRemove() 超出屏幕移除该对象
